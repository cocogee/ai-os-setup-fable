# 04_knowledge — Your second brain library

**Type:** Static context (grows, rarely edited).

**Belongs here:**
- `notes/` — your thinking, brainstorms, grill-me session outputs
- `transcripts/` — meeting/video/call transcripts
- `research/` — research dumps and findings
- `references/` — docs, guides, things you cite

**Does not belong here:** Live data, credentials, project work, outputs.

**Before modifying:** Knowledge files are mostly append-only. Don't rewrite a note; add to it or create a dated new one. If something is obsolete, move it to 10_archive.
