# 01_profile — Who you are

**Type:** Static context (changes rarely).

**Belongs here:** Background, preferences, goals, and a log of big decisions. Claude reads these so it sounds like a teammate, not a stranger.

**Does not belong here:** Business details (02_business), daily tasks, or anything that changes weekly.

**Before modifying:** Show the user what will change. Never overwrite — append or revise with the old version noted in 10_archive if the change is large.
