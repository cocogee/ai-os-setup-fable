# 08_logs — What happened

**Type:** Logs (append-only).

- `session_logs/` — short summary at the end of significant work sessions
- `decision_logs/` — system/architecture decisions (personal decisions go in 01_profile/DECISIONS.md)
- `error_logs/` — mistakes, what caused them, what was fixed so they don't repeat
- `automation_logs/` — every automated run, once automations exist

**Before modifying:** Logs are append-only. Never edit or delete past entries; add corrections as new entries.
