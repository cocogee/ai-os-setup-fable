# 05_connections — Live data sources

**Type:** LIVE CONNECTIONS. Higher risk than everything else in this OS.

**Belongs here:**
- `CONNECTIONS_INDEX.md` — the registry of every live connection
- `api_docs/` — saved API/CLI documentation for tools you connect
- `credentials_notes/` — notes ABOUT keys (where stored, what scope). **NEVER the actual keys or secrets.**
- `test_logs/` — results of manual connection tests

**Does not belong here:** Actual API keys, passwords, tokens. Keys live in your system's environment/keychain, never in markdown.

**Before modifying:** Read 00_system/SAFETY_RULES.md. No new connection without explicit user approval. Every connection gets an entry in CONNECTIONS_INDEX.md listing what it can physically do.
