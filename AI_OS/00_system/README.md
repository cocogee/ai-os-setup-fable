# 00_system — The OS rulebook

**Type:** Static context (rules about the system itself).

**Belongs here:** The master index, model-routing rules, safety rules, and changelog. Files that describe HOW the OS works.

**Does not belong here:** Your personal info (01_profile), business info (02_business), or any project work.

**Before modifying:** Read the file first. Changes here affect every future session, so explain any change to the user (Teaching Mode) and log it in CHANGELOG.md.
