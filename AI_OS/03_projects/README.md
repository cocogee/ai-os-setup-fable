# 03_projects — Active work

**Type:** Working area (changes constantly).

**Belongs here:** One folder per project inside `active/`. Each project folder starts with a BRIEF.md (use 06_skills/create_project_brief.md). Move to `paused/` or `completed/` as status changes — move, don't delete.

**Does not belong there:** Reference knowledge (04_knowledge), finished deliverables for outside use (09_outputs).

**Before modifying:** Check whether the project folder already exists before creating a new one. Read its BRIEF.md before working on it.
