# 09_outputs — Finished work products

**Type:** Outputs.

- `drafts/` — work in progress
- `reports/` — weekly reviews, analyses
- `exports/` — data pulled out for other tools
- `deliverables/` — final, shippable artifacts

**Does not belong here:** Source context or knowledge — outputs are RESULTS, derived from the rest of the OS.

**Before modifying:** Never overwrite a deliverable; save a new version (v2, v3) or move the old one to 10_archive/old_versions.
