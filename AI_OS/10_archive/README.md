# 10_archive — Nothing is deleted, things are retired

**Type:** Archive.

- `old_versions/` — superseded versions of files
- `deprecated/` — material no longer accurate or in use
- `raw_imports/` — messy material from a pre-existing system, kept as-is until sorted

**Rule:** Archive instead of delete, always. When archiving, add a line to 00_system/CHANGELOG.md saying what moved and why.

**Before modifying:** Archive is read-mostly. Pull things OUT if needed; don't edit them in place.
